La etiqueta video: Para especificar la ubicacion del archivo que se reproducira, puede usar el elemento <source>dentro del elemento <video>.(Esto debe reemplazar el atributo src en la etiqueta<video> de apertura.)Tambien puede utilizar varios elementos <source>para especificar que el video esta disponible en diferentes formatos.(Debido a un error en el iPad, debe propocionarse el video MP4como el primer formato. De lo contrario puede que no inicie.)

src:este atributo especifica la ruta de acceso al video.

type:se debe utilizar este atributo para decirle al navegador que el formato es el video. De lo contrario,descargara parte del video para ver si puede reproducir el archivo(que tomara tiempo y ancho de banda).

codecs: el codec que uso para codificar el video se suministra dentro del atributo type.tenga en cuenta el uso den comillas simples,asi como comillas dobles en el atributo de tipo,cuando se suministra

Text Peoperties

Este ejemplo combina muchas de las técnicas mostradas en los ejercicios

hay propiedades para controlar el tipo de fuente, el tamaño, la ubicación, el estilo y el espaciado.

Características

Puede controlar el espacio entre las líneas de texto, las letras individuales y las palabras.El texto también se puede alinear a la izquierda, derecha,centrada o justificada.Tambíen puede tener sangría.

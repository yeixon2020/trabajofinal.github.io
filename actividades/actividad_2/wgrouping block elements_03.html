El elemento <div>permite agrupar un conjunto de elementos juntos en el cuadro de nivel de bloque.Por ejemplo,puede crear un elemento <div>para que contengan todos los elementos del encabezado del sitio (el logotipo y la navegacion), o puede crear un elemento <div>para que contenga informacion en bloques.

En jun navegador, el contenido del elemento no harea ninguna diferencia en la presentacion de la pagina.Sin embargo,el uso de un atributo ID o Class en el elemento <div>significa que puedes crear reglas de estilo CSS para indicar cuanto espacio debe ocuparel elemento <div> en la pantalla y cambiar la apariencia de todos los elementos contenidos en elTambien puede facilitar el seguimiento del codigo si ha utilizado elentos <div>para contener cada secion de la pagina.

Dado que puede haber varios otros elementos dentro del elemento<div>,puede ser util agregra un comentario despues de la etiqueta</div>de cierre.Esto le permite ver claramente a que etiqueta de apertura se supone que corresponde.

Background Color- Color de Fondo

Si no especifica un color de fondo, el fondo es transparente.

CSS trata cada elemnto HTML como un cuadro y la propiedad backgrround-color establece el colorf de fondo para el cuadro.Se puede especificar un color de fondo de las mismas tres formas en que puede especificar colores de primer plano:Valores RGB, código hexagesimal y nombre de colores.tambíen hemos utilizado la propiedad padding para separa el texto de los bordes de los cuadros.Esto hace que sea más fácil de leer.
